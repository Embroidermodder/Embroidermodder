\doxysubsection{Detailed Description}
Clients can handle the \textquotesingle{}done\textquotesingle{} event to get notified when the related request is done.

Note, because wl\+\_\+callback objects are created from multiple independent factory interfaces, the wl\+\_\+callback interface is frozen at version 1.
