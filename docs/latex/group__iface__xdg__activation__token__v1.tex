\doxysubsection{Detailed Description}
An object for setting up a token and receiving a token handle that can be passed as an activation token to another client.

The object is created using the xdg\+\_\+activation\+\_\+v1.\+get\+\_\+activation\+\_\+token request. This object should then be populated with the app\+\_\+id, surface and serial information and committed. The compositor shall then issue a done event with the token. In case the request\textquotesingle{}s parameters are invalid, the compositor will provide an invalid token.
