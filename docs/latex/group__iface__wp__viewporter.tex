\doxysubsection{Detailed Description}
The global interface exposing surface cropping and scaling capabilities is used to instantiate an interface extension for a wl\+\_\+surface object. This extended interface will then allow cropping and scaling the surface contents, effectively disconnecting the direct relationship between the buffer and the surface size.
