\doxysubsection{Detailed Description}
A singleton global object that provides support for shared memory.

Clients can create wl\+\_\+shm\+\_\+pool objects using the create\+\_\+pool request.

On binding the wl\+\_\+shm object one or more format events are emitted to inform clients about the valid pixel formats that can be used for buffers.
