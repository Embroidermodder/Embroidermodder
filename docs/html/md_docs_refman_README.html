Build

To build, run: 

$ latexmk -pdf

to clean up files after the build process run: 

$ latexmk -c

Dependencies on Ubuntu

On Ubuntu, this line should get the necessary packages (FIX THIS): 

$ sudo apt install texlive texlive-latex-base latexmk
