
\chapter{Libembroidery API}

\section{EmbVector}

The basic vector type for libembroidery is:

\begin{lstlisting}
typedef struct EmbVector_ {
    float x;
    float y;
} EmbVector;
\end{lstlisting}

Equivalent to:

\begin{equation}
    \begin{pmatrix}
    a_x \\ a_y
    \end{pmatrix}
\end{equation}

\subsection{embVector\_add(EmbVector a, EmbVector b) $\to$ EmbVector}
\index{\texttt{embVector\_add}}

Takes two arguments, both EmbVectors and adds them together, returning the result.

Equivalent to:

\begin{equation}
    \begin{pmatrix}
    c_x \\ c_y
    \end{pmatrix} =
    \begin{pmatrix}
    a_x \\ a_y
    \end{pmatrix} +
    \begin{pmatrix}
    b_x \\ b_y
    \end{pmatrix}
\end{equation}

\subsection{embVector\_subtract(EmbVector a, EmbVector b) $\to$ EmbVector}
\index{\texttt{embVector\_subtract}}

Takes two arguments, both EmbVectors and adds them together, returning the result.

Equivalent to:

\begin{equation}
    \begin{pmatrix}
    c_x \\ c_y
    \end{pmatrix} =
    \begin{pmatrix}
    a_x \\ a_y
    \end{pmatrix} -
    \begin{pmatrix}
    b_x \\ b_y
    \end{pmatrix}
\end{equation}
