Copyright 2018-2022 The Embroidermodder Team Licensed under the terms of the zlib license.

This file contains all the read and write functions for the library.

Thanks to Jason Weiler for describing the binary formats of the HUS and VIP formats at:

http://www.jasonweiler.com/HUSandVIPFileFormatInfo.html

Further thanks to github user tatarize for solving the mystery of the compression in:

https://github.com/EmbroidePy/pyembroidery

with a description of that work here:

https://stackoverflow.com/questions/7852670/greenleaf-archive-library

This is based on their work.
