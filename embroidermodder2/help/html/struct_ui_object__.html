Detailed Description

This covers the inbuilt designs: Dolphin, Snowflake and Heart. Covers Rotate, Scale and Point UI events.

This was an idea for storing the current command state: could be combined with EmbView since you can't have more than one active command. If a command calls a sub command it will store the position in the parents.
